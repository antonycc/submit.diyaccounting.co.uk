What is MCP?

The Model Context Protocol (MCP) is an open standard for connecting AI applications to external data sources, tools, and workflows. Think of MCP like a USB-C port for AI applications — just as USB-C provides a standardised way to connect devices, MCP provides a standardised way to connect AI assistants to external systems.

An MCP server for DIY Accounting Submit will allow AI assistants such as Claude, ChatGPT, and others to interact with your VAT submissions, view obligations, and manage returns on your behalf — all with your explicit consent and control.

Resources

Expose your VAT obligations, return history, and submission receipts as structured data that AI assistants can read and reason about.

Tools

Let AI assistants execute actions like retrieving obligations from HMRC, preparing VAT returns, and checking submission status — with your approval at every step.

Security & Consent

MCP requires explicit user consent for all data access and tool invocations. Your credentials are never shared with the AI model. You remain in full control of what actions are performed.
